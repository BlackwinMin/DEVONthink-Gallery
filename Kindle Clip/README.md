# DEVONthink Kindle Clip 摘抄

摘抄应易读，本文用 Shortcuts（快捷指令）将 Kindle 硬件中的书摘导成 Markdown 文档，既可以融入 DEVONthink 阅读摘抄体系，也可以自由选择其他管理方式。

使用方式请阅读原文。
